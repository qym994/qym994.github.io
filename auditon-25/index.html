防抖和节流​

防抖和节流都是为了阻止操作高频触发，从而浪费性能

防抖：连续触发事件但在一定时间内只执行最后一次
代码实现(利用闭包)：

function debounce(callback,delay){
let timer = null
return function(){
  clearTimeout(timer)
  timer = setTimeout(()=>{
    callback.apply(this,arguments)
  },delay)
}

}

节流：当持续触发事件时，保证一定时间段内只调用一次事件处理函数
代码实现(利用闭包)： 

function throttle(callback,delay){
  let flag = true
  return function(){
    if(flag){
      setTimeout(()=>{
        callback.apply(this,arguments)
        flag = true
      },delay)
      flag = false
    }
  }
}
